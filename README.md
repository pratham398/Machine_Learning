# Machine_Learning
Heart disease detection

📌 Project Overview

This project predicts the risk of heart disease using Machine Learning techniques in Python. The model is trained using medical and demographic data to determine whether a person is likely to develop coronary heart disease within the next 10 years.

The goal of this project is to perform data preprocessing, analysis, and predictive modeling to identify patterns that contribute to heart disease risk.

📊 Dataset

The dataset used in this project is the Framingham Heart Study dataset, which contains medical attributes related to heart disease risk.

Some important features in the dataset include:

Age
Gender
Smoking status
Blood Pressure
Cholesterol level
Diabetes
Body Mass Index (BMI)
Heart Rate
Glucose level
10-year risk of coronary heart disease (Target Variable)

⚙️ Project Workflow
1. Data Collection

The dataset is loaded and inspected using Pandas.

2. Data Exploration

Exploratory Data Analysis (EDA) is performed to understand:

Dataset structure
Missing values
Statistical summaries
3. Data Preprocessing

Data preprocessing includes:

Handling missing values
Feature selection
Splitting the dataset into training and testing sets
4. Feature Scaling

Standardization is applied using StandardScaler to normalize feature values.

5. Model Training

A Logistic Regression model is trained on the dataset to predict the likelihood of heart disease.

6. Model Evaluation

The model performance is evaluated using accuracy scores on training and testing data.
